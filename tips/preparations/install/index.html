Rubyのインストール

nanocはRuby製のソフトウェアです。nanocを利用するには最初にRubyをインストールする必要があります。 例えば、あなたの使用しているOSがWindowsでしたら、RubyInstaller for Windowsを使うという方法があります。配布先のページは英語ページですが、Ruby Associationのサイト上で日本語による詳細なRubyInstaller for Windowsを利用したRubyのインストール方法について解説「Rubyアソシエーション: インストール手順(Windows)」があります。

Rubyのインストールについては、OSごとの違いがあったり、Rubyのバージョンを管理するソフトを使用する場合もあったりと、いろいろな方法があります。

nanocのインストール

nanocのインストールにはRubyGemsというパッケージ管理システムを利用します。Rubyのバージョンが1.9以降であれば、Rubyをインストールすれば使えるようになっています。そして、RubyGemsは「gem」というコマンドで実行できます。

例えば今回の場合、nanocをRubyGemsを利用してインストールしますので、Windowsですとコマンドプロンプトを立ち上げて次のコマンドを実行します。

gem install nanoc

筆者の実行環境（Windows XP、Ruby1.9.3p385）においては文字コード関係のエラーが発生しました。一つの解決策として、この場合は事前に次のコマンドを実行しておくべきでした。

set LANG=ja_JP.UTF-8

再度インストールをした所、今度は無事に完了しました。また、nanocの利用時にはコンパイルを行いますが、その際にも文字コードの問題に悩まされます。 今回はコマンドプロンプト上でLANGという環境変数を一時的に設定してインストールを完了しましたが、一々環境変数を設定するのは面倒です。この面倒はOSの環境変数の設定を行うことで解決します。

他のOSの利用を（やや強めに）おすすめします

このような文字コードの問題を解決する方法としては、他のOSを利用するという手段があります。 他のOSというと敷居が高そうに感じますが、無料で、Windows上の仮想環境で利用できるOSがあります。例えばUbuntuの利用はいかがでしょうか。（※仮想環境での他OSの実行は、ある程度マシンスペックが必要です。）
